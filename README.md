# MusicParser

Live workspace for HTML / JS / Python experiments.

## Run locally

```bash
python3 server.py
```

Open:

```text
http://127.0.0.1:8000
```

## Files

- `index.html` - browser UI
- `app.js` - editable parser logic
- `style.css` - UI styles
- `server.py` - Python static server with a small save endpoint

## Notes

GitHub Pages can show the static HTML/JS UI, but it cannot run the Python save endpoint. For saving from the browser, run `server.py` locally or in a real Python host/Codespace.
